# A Tour of Kestrel

Three short programs that build up a feel for the language. Each is follow-along — you copy, run, and see output before moving on. Each adds one new idea per step. ~80-120 lines final.

## Tours

- [Text Adventure](text-adventure.md) — rooms, exits, items. Covers structs, enums with payloads, dictionaries, Optional.
- [Wizard Duel](wizard-duel.md) — two wizards trade spells. Covers recursive enums, mutating methods, pattern matching with payloads.
- [Turtle Graphics](turtle-graphics.md) — drive a turtle around an ASCII grid. Covers closures, recursive command programs, protocols.

Read them in order. Each one's mental model carries forward, but every tour stands on its own.
