## Description

For Loops allow looping through an iterator or iterable

## Use Case

For loops are an idiomatic concept in other languages, simplifying control flow and enabling much more declarative programming

```kestrel
module Main

func main() {
    let foobars = ["Foo", "Bar", "FooBar"];

    for foo in foobars {
        print(foo);
    }
}
```

## Proposed Solution

Rather than being their own construct, for loops will be pure syntactic sugar. The syntax for loops is as following:

```kestrel
for pattern in iterable {
    // code goes here
}
```

This desugars to:

```
var iterator = iterable.iter();

while let .Some(pattern) = iterator.next() {
    // code goes here
}
```
