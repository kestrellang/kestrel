# Issue #14: Compound Assignment Operators

**Author:** @jkpdino
**Created:** 2026-01-22T20:53:35Z
**URL:** https://github.com/jkpdino/kestrel/issues/14

## Description

## Description

Operators like +=, -=

## Use Case

This feature simplifies the intent behind doing an operation on a value, to itself

```kestrel
module Main

x = x + 1
```

## Proposed Solution (Optional)

```
@builtin(.AddAssignableOperator)
protocol AddAssignable[T] {
    mutating func addAssign(other: T) 
}

extend Add[T]: AddAssignable[T] where Add.Output = Self {
    mutating func addAssign(other: T) {
        self = self + other;
    }
}
```
