## Description

## Description

Add character literal syntax using single quotes (`'a'`) to represent Unicode scalar values.

## Use Case

Currently, working with characters requires verbose constructs like `CodePoint(UInt32(intLiteral: 65))` for the letter 'A'.

```kestrel
// Proposed syntax
let a = 'a'
let newline = '\n'
let emoji = '🦅'
let unicode = '\u{1F600}'
```

## Proposed Solution

- Lexer tokenizes single-quoted literals
- Semantic layer validates content and escape sequences
- Integrate with `ExpressibleByCharLiteral` protocol
- Default type: `lang.i32`
